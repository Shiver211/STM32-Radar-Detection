# 图片资源需求清单

当前目录下的 PNG 文件均为 1x1 空白占位图，用于保证工程结构完整并可正常加载。

## 1. hero_banner.png

- 文件路径：resources/images/hero_banner.png
- 建议尺寸：1440x320
- 画面内容：
  - 横向科技感主视觉横幅
  - 左侧可出现雷达扫描扇形、心电线条、呼吸波纹
  - 右侧留出浅色透明区域用于叠加标题文字
- 风格关键词：医疗监护、科技蓝绿、低噪点、层次光晕
- 用途：主界面顶部横幅背景

## 2. icon_heart.png

- 文件路径：resources/images/icon_heart.png
- 建议尺寸：128x128（也可导出 64x64）
- 画面内容：
  - 简洁心脏图标，建议线框+轻微发光
  - 需适配浅色背景
- 风格关键词：圆角、扁平半拟物、医监图标
- 用途：心率区域标题或按钮前缀图标

## 3. icon_breath.png

- 文件路径：resources/images/icon_breath.png
- 建议尺寸：128x128（也可导出 64x64）
- 画面内容：
  - 呼吸气流/肺部抽象图标
  - 与心率图标保持统一视觉语言
- 风格关键词：柔和线条、方向感、呼吸节律
- 用途：呼吸区域标题或按钮前缀图标

## 4. bg_texture.png

- 文件路径：resources/images/bg_texture.png
- 建议尺寸：1920x1080
- 画面内容：
  - 低对比度纹理底图（纸质肌理或轻微网格）
  - 亮度均匀，避免影响文字可读性
- 风格关键词：高级、克制、浅暖色调
- 用途：整页背景纹理叠加
